# Z

A simple React Native / Expo personal finance app (รายรับ-รายจ่าย) — tracks transactions and categories with a swipeable modal UI.

## Features

- Add / edit / delete transactions
- Custom categories (income/expense)
- Daily progress circle and simple summary
- Swipeable, draggable modal for adding/editing items

## Quick start (development)

Prerequisites:
- Node.js (16+ recommended)
- Expo CLI (optional) or use `npx expo`

Install dependencies:

```bash
cd D:\User\Documents\coding\Z
npm install
# or
# yarn install
```

Run the app (Expo):

```bash
npx expo start --port 3000
```

Open on device/emulator using the Expo dev tools.

## Repo layout

- `App.tsx` — main application UI
- `src/` — database module, components, screens
